# 16-bit RISC Processor Design using Verilog HDL

## Overview

This project implements a simplified 16-bit RISC Processor using Verilog HDL.

The processor demonstrates the basic architecture of a CPU including instruction fetch, register operations, arithmetic and logic operations, and memory access.

---

## Features
- 16-bit datapath
- Program Counter
- Instruction Memory
- Register File
- ALU
- Control Unit
- Data Memory
- Multiplexer
- CPU Top Module
- Testbench Simulation
